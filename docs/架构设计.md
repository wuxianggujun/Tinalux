# Tinalux 架构设计

## 架构总览

```
              ┌→ Platform ──→ Core
              │
App (胶水) ───┼→ Rendering ──→ Core + Skia
              │
              └→ UI ─────────→ Core + Skia
```

5 个模块。App 是唯一的胶水层，其他模块**互不依赖**。

| 模块 | 职责 | 依赖 | 不依赖 |
|------|------|------|--------|
| **Core** | 事件类型、日志、数学工具 | 仅标准库 | — |
| **Platform** | 窗口抽象（创建、事件、GL 上下文） | Core + GLFW(桌面)/NDK(Android) | Skia |
| **Rendering** | Skia GPU 上下文、Surface 创建 | Core + Skia | Platform, GLFW |
| **UI** | Widget 树、布局、控件、主题 | Core + Skia | Rendering, Platform |
| **App** | 组装一切、驱动主循环 | Platform + Rendering + UI + Core | — |

**关键约束**：
- Rendering **不 include 任何 Platform 头文件**，通过函数指针参数接收 GL proc address
- UI **不 include 任何 Rendering 头文件**，只面向 SkCanvas 和 core::Event
- 只有 App 知道所有模块的存在

## 渲染管线

```
App::init()
│
├─ window_ = platform::createWindow(config)
├─ auto getProc = window_->glGetProcAddress()
└─ grContext_ = rendering::createGLContext(getProc)

App::run() 每帧循环
│
├─ window_->pollEvents()
├─ 如果 needsRedraw_:
│   ├─ auto surface = rendering::createWindowSurface(grContext_, w, h)
│   ├─ SkCanvas* canvas = surface->getCanvas()
│   ├─ rootWidget_->measure(Constraints::tight(w, h))   ← 布局第一遍
│   ├─ rootWidget_->arrange({0, 0, w, h})               ← 布局第二遍
│   ├─ rootWidget_->draw(canvas)                         ← 绘制
│   ├─ rendering::flushFrame(grContext_)
│   └─ window_->swapBuffers()
└─ needsRedraw_ = false
```

## 一、布局系统（参考 Flutter / Druid）

### 核心模型：Constraints 约束协商

```
父组件
  │ 传下 Constraints{minW, maxW, minH, maxH}
  ▼
子组件::measure(constraints)
  │ 在约束范围内决定自己的尺寸
  │ 返回 SkSize
  ▼
父组件
  │ 收到所有子组件的 SkSize
  │ 运行布局算法（Flex/VBox/HBox）
  ▼
父组件::arrange()
  │ 给每个子组件分配最终 bounds
  ▼
子组件::arrange(bounds)
  │ 递归继续
```

### API

```cpp
namespace tinalux::ui {

struct Constraints {
    float minWidth = 0;
    float maxWidth = std::numeric_limits<float>::infinity();
    float minHeight = 0;
    float maxHeight = std::numeric_limits<float>::infinity();

    // 工厂方法
    static Constraints tight(float w, float h);         // min == max
    static Constraints loose(float maxW, float maxH);   // min = 0
    static Constraints unbounded();                      // 全 infinity

    // 工具
    Constraints withMaxWidth(float w) const;
    Constraints withMaxHeight(float h) const;
    SkSize constrain(SkSize size) const;  // 夹到范围内
    bool isTight() const;
};
```

### 两遍布局流程

```cpp
class Widget {
    // 第一遍：measure（自底向上）
    // 在给定约束范围内，返回自己想要的尺寸
    virtual SkSize measure(const Constraints& constraints) = 0;

    // 第二遍：arrange（自顶向下）
    // 父组件分配最终位置
    virtual void arrange(const SkRect& bounds);
};
```

### 布局管理器

```cpp
class Layout {
public:
    virtual ~Layout() = default;

    // 由 Container 调用：
    // 1. 给每个子组件 measure
    // 2. 根据算法分配位置
    virtual SkSize measure(const Constraints& constraints,
                           std::vector<std::shared_ptr<Widget>>& children) = 0;
    virtual void arrange(const SkRect& bounds,
                         std::vector<std::shared_ptr<Widget>>& children) = 0;
};

class VBoxLayout : public Layout {
public:
    float spacing = 0;
    float padding = 0;

    // measure: 宽取最大子组件宽，高取所有子组件高之和 + spacing
    // arrange: 从上往下逐个放置
    SkSize measure(const Constraints& constraints,
                   std::vector<std::shared_ptr<Widget>>& children) override;
    void arrange(const SkRect& bounds,
                 std::vector<std::shared_ptr<Widget>>& children) override;
};

class HBoxLayout : public Layout {
    // 同理，横向排列
};
```

### Container 如何驱动布局

```cpp
class Container : public Widget {
    SkSize measure(const Constraints& constraints) override {
        if (layout_) {
            return layout_->measure(constraints, children_);
        }
        // 无 layout 时：取第一个子组件的尺寸
        if (!children_.empty()) {
            return children_[0]->measure(constraints);
        }
        return constraints.constrain({0, 0});
    }

    void arrange(const SkRect& bounds) override {
        bounds_ = bounds;
        if (layout_) {
            layout_->arrange(bounds, children_);
        }
    }
};
```

## 二、事件系统（参考 Chromium / GTK4）

### 三阶段分发：捕获 → 目标 → 冒泡

```
鼠标点击 (300, 200)
    │
    ├─ 第一步：命中测试（找目标）
    │   root->hitTest(300, 200)
    │   递归找到最前面（Z 序最高）的叶子 Widget
    │   得到：eventPath = [root, panel, button]
    │
    ├─ 第二步：捕获阶段（根 → 目标）
    │   root.onEventCapture(click) → 不拦截
    │   panel.onEventCapture(click) → 不拦截
    │
    ├─ 第三步：目标阶段
    │   button.onEvent(click) → handled = true → 停止
    │
    └─ 第四步：冒泡阶段（目标 → 根，仅当未 handled）
        button → panel.onEvent(click) → root.onEvent(click)
```

### API

```cpp
namespace tinalux::core {

enum class EventType {
    None,
    WindowClose, WindowResize, WindowFocus,
    KeyPress, KeyRelease, KeyRepeat,
    MouseButtonPress, MouseButtonRelease,
    MouseMove, MouseScroll,
    TextInput,
};

class Event {
public:
    virtual ~Event() = default;
    virtual EventType type() const = 0;

    bool handled = false;        // 设为 true 停止后续传播
    bool stopPropagation = false; // 设为 true 阻止冒泡
};

// 具体事件类型
class WindowResizeEvent : public Event { /* width, height */ };
class KeyEvent : public Event { /* key, scancode, mods, type_ */ };
class MouseMoveEvent : public Event { /* x, y */ };
class MouseButtonEvent : public Event { /* button, mods, x, y, type_ */ };
class MouseScrollEvent : public Event { /* xOffset, yOffset */ };
class TextInputEvent : public Event { /* codepoint (uint32_t) */ };

}  // namespace tinalux::core
```

### Widget 事件接口

```cpp
class Widget {
    // 捕获阶段（从根往目标走时调用）
    // 返回 true = 拦截，不再往下传
    virtual bool onEventCapture(core::Event& event) { return false; }

    // 目标 + 冒泡阶段
    // 返回 true = 已消费
    virtual bool onEvent(core::Event& event) { return false; }

    // 命中测试：点 (x,y) 是否落在自己身上
    virtual Widget* hitTest(float x, float y);
};
```

### 事件分发器（在 App 中实现）

```cpp
void Application::dispatchEvent(core::Event& event) {
    if (!rootWidget_) return;

    // 鼠标事件需要命中测试
    if (isMouseEvent(event)) {
        auto [x, y] = getMousePos(event);
        Widget* target = rootWidget_->hitTest(x, y);
        if (!target) return;

        // 构建事件路径：root → ... → target
        std::vector<Widget*> path = buildEventPath(target);

        // 捕获阶段（正序，不含 target）
        for (int i = 0; i < path.size() - 1; i++) {
            if (path[i]->onEventCapture(event)) return;
        }

        // 目标阶段
        target->onEvent(event);
        if (event.handled || event.stopPropagation) return;

        // 冒泡阶段（反序，不含 target）
        for (int i = path.size() - 2; i >= 0; i--) {
            path[i]->onEvent(event);
            if (event.handled || event.stopPropagation) return;
        }
    }

    // 键盘事件发给焦点 Widget
    if (isKeyEvent(event) && focusedWidget_) {
        focusedWidget_->onEvent(event);
    }
}
```

### 焦点管理

## 三、渲染系统（参考 Flutter / LVGL）

### 脏标记 + 按需重绘

```
Button 文字变了
    │
    ├─ button.markDirty()
    │   └─ dirty_ = true
    │   └─ 向上通知：parent_->onChildDirty()
    │       └─ ... 一直传到 root
    │       └─ App::needsRedraw_ = true
    │
    └─ 下一帧 App::run() 检查 needsRedraw_
        └─ true → 重绘整棵树
        └─ false → 跳过这一帧（省 GPU）
```

### Widget 绘制接口

```cpp
class Widget {
    // 绘制自身。canvas 坐标已变换到 widget 本地空间。
    virtual void onDraw(SkCanvas* canvas) = 0;

    // 框架调用此方法（处理坐标变换和裁剪）
    void draw(SkCanvas* canvas) {
        if (!visible_) return;
        canvas->save();
        canvas->translate(bounds_.x(), bounds_.y());
        canvas->clipRect(SkRect::MakeWH(bounds_.width(), bounds_.height()));
        onDraw(canvas);
        canvas->restore();
    }

    // 标脏
    void markDirty();

protected:
    bool dirty_ = true;
    Widget* parent_ = nullptr;  // 用于向上传播脏标记
};
```

### Container 递归绘制

```cpp
class Container : public Widget {
    void onDraw(SkCanvas* canvas) override {
        // 先画自己的背景（如果有）
        drawBackground(canvas);

        // 再按 Z 序画子组件（先添加的在下层）
        for (auto& child : children_) {
            child->draw(canvas);
        }
    }
};
```

## 四、Widget 接口总览（参考 Druid）

5 个核心方法，职责清晰：

```cpp
namespace tinalux::ui {

class Widget {
public:
    virtual ~Widget() = default;

    // ── 布局 ──
    virtual SkSize measure(const Constraints& constraints) = 0;
    virtual void arrange(const SkRect& bounds);

    // ── 事件 ──
    virtual bool onEventCapture(core::Event& event) { return false; }
    virtual bool onEvent(core::Event& event) { return false; }
    virtual Widget* hitTest(float x, float y);

    // ── 绘制 ──
    virtual void onDraw(SkCanvas* canvas) = 0;
    void draw(SkCanvas* canvas);  // 框架调用，处理 save/translate/clip/restore

    // ── 属性 ──
    SkRect bounds() const;
    bool visible() const;
    void setVisible(bool v);
    bool focused() const;
    void setFocused(bool f);
    virtual bool focusable() const { return false; }

    // ── 脏标记 ──
    void markDirty();
    bool isDirty() const;

protected:
    SkRect bounds_ = SkRect::MakeEmpty();
    bool visible_ = true;
    bool focused_ = false;
    bool dirty_ = true;
    Widget* parent_ = nullptr;
};

}  // namespace tinalux::ui
```

## 八、App 模块 API

**唯一的胶水层。连接 Platform、Rendering、UI。**

```cpp
namespace tinalux::app {

class Application {
public:
    Application() = default;
    ~Application();

    Application(const Application&) = delete;
    Application& operator=(const Application&) = delete;

    bool init(const platform::WindowConfig& config = {});
    int run();
    void shutdown();

    void setRootWidget(std::shared_ptr<ui::Widget> root);
    platform::Window* window() const;

private:
    void onEvent(core::Event& event);
    void dispatchEvent(core::Event& event);  // 三阶段分发
    void renderFrame();

    std::unique_ptr<platform::Window> window_;
    sk_sp<GrDirectContext> grContext_;
    std::shared_ptr<ui::Widget> rootWidget_;
    ui::Widget* focusedWidget_ = nullptr;
    bool needsRedraw_ = true;
};

}  // namespace tinalux::app
```

**使用示例**：

```cpp
int main() {
    tinalux::app::Application app;
    if (!app.init({.width = 960, .height = 640, .title = "Tinalux"})) return 1;

    auto root = std::make_shared<tinalux::ui::Panel>();
    root->setBackgroundColor(tinalux::ui::currentTheme().background);

    auto vbox = std::make_unique<tinalux::ui::VBoxLayout>();
    vbox->spacing = 12;
    vbox->padding = 24;
    root->setLayout(std::move(vbox));

    auto label = std::make_shared<tinalux::ui::Label>("Hello, Tinalux!");
    label->setFontSize(32);
    root->addChild(label);

    auto btn = std::make_shared<tinalux::ui::Button>("Click Me");
    btn->onClick([] { /* ... */ });
    root->addChild(btn);

    app.setRootWidget(root);
    return app.run();
}
```
