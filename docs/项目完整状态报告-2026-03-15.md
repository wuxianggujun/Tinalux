## 🏗️ 架构概览

### 模块结构
```
┌─────────────────────────────────────────────┐
│              App (应用层)                     │
│  - Application, UIContext, Window管理        │
└─────────────────────────────────────────────┘
         ↓              ↓              ↓
┌──────────────┐ ┌──────────────┐ ┌──────────────┐
│   Platform   │ │      UI      │ │  Rendering   │
│   (平台层)    │ │   (UI层)     │ │  (渲染层)     │
│  - GLFW      │ │  - Widget    │ │  - Canvas    │
│  - Window    │ │  - Layout    │ │  - Skia封装   │
│  - Events    │ │  - Theme     │ │  - GPU渲染    │
└──────────────┘ └──────────────┘ └──────────────┘
         ↓              ↓              ↓
┌─────────────────────────────────────────────┐
│              Core (核心层)                    │
│  - Geometry, Color, Event, Logger            │
└─────────────────────────────────────────────┘
```

### 设计原则
1. **星形依赖**：App作为中心，其他模块独立
2. **Skia隔离**：公共API不暴露Skia类型
3. **内存安全**：unique_ptr/shared_ptr管理Widget树
4. **事件三阶段**：capture→target→bubble
5. **两遍布局**：measure→arrange
