## 📌 概述

主题系统是现代UI框架的核心功能，决定了应用的视觉风格和用户体验。

### 当前状态 ⚠️

**已有功能**：
- ✅ 基础Theme结构（颜色定义）
- ✅ dark/light主题切换
- ✅ 全局主题访问

**问题**：
- ❌ 只有全局颜色，无法定制单个控件
- ❌ 无样式继承机制
- ❌ 无伪类样式（:hover, :pressed, :disabled）
- ❌ 主题切换无动画
- ❌ 无主题持久化

---

## 一、设计目标

### 1.1 核心需求

1. **分层样式系统**
   - 全局主题（Theme）
   - Widget类型样式（ButtonStyle, TextInputStyle）
   - Widget实例样式（单个控件自定义）

2. **样式继承**
   - 子Widget继承父Widget样式
   - 支持样式覆盖

3. **状态样式**
   - 正常（normal）
   - 悬停（hover）
   - 按下（pressed）
   - 禁用（disabled）
   - 聚焦（focused）

4. **主题切换**
   - 平滑动画过渡
   - 保存用户偏好
   - 运行时动态切换

5. **易用性**
   - 简洁的API
   - 合理的默认值
   - 类型安全

## 二、架构设计

### 2.1 类层次结构

```
Theme (全局主题)
  ├─ ColorScheme (配色方案)
  ├─ Typography (字体排版)
  ├─ Spacing (间距规范)
  └─ WidgetStyles (控件样式集合)
      ├─ ButtonStyle
      ├─ TextInputStyle
      ├─ CheckboxStyle
      └─ ...

Widget
  ├─ style_ (实例样式，可选)
  └─ resolveStyle() (解析最终样式)
```

### 2.2 样式解析优先级

```
实例样式 > 类型样式 > 主题默认样式
```

### 3.4 WidgetStyle（控件样式基类）

```cpp
// include/tinalux/ui/WidgetStyle.h

namespace tinalux::ui {

enum class WidgetState {
    Normal,
    Hovered,
    Pressed,
    Disabled,
    Focused
};

// 状态样式（支持多状态组合）
template<typename T>
struct StateStyle {
    T normal;
    std::optional<T> hovered;
    std::optional<T> pressed;
    std::optional<T> disabled;
    std::optional<T> focused;
    
    // 根据状态解析值
    const T& resolve(WidgetState state) const {
        switch (state) {
        case WidgetState::Hovered:
            return hovered.value_or(normal);
        case WidgetState::Pressed:
            return pressed.value_or(hovered.value_or(normal));
        case WidgetState::Disabled:
            return disabled.value_or(normal);
        case WidgetState::Focused:
            return focused.value_or(normal);
        default:
            return normal;
        }
    }
};

}  // namespace tinalux::ui
```

## 四、Widget集成

### 4.1 Widget基类扩展

```cpp
// include/tinalux/ui/Widget.h

class Widget {
public:
    // 设置实例样式（可选）
    template<typename StyleT>
    void setStyle(const StyleT& style) {
        customStyle_ = std::make_shared<StyleT>(style);
        markPaintDirty();
    }
    
    // 获取实例样式
    template<typename StyleT>
    const StyleT* style() const {
        return dynamic_cast<const StyleT*>(customStyle_.get());
    }
    
    // 解析最终样式（优先级：实例 > 主题）
    template<typename StyleT>
    StyleT resolveStyle() const {
        // 1. 检查实例样式
        if (auto* custom = style<StyleT>()) {
            return *custom;
        }
        
        // 2. 使用主题默认样式
        return getThemeStyle<StyleT>();
    }
    
protected:
    // 当前状态
    WidgetState currentState() const;
    
private:
    std::shared_ptr<void> customStyle_;  // 类型擦除
};
```

### 4.2 Button使用样式

```cpp
// src/ui/widgets/Button.cpp

void Button::onDraw(rendering::Canvas& canvas)
{
    // 解析样式
    const ButtonStyle style = resolveStyle<ButtonStyle>();
    
    // 获取当前状态
    const WidgetState state = currentState();
    
    // 根据状态获取颜色
    const core::Color bgColor = style.backgroundColor.resolve(state);
    const core::Color textColor = style.textColor.resolve(state);
    const core::Color borderColor = style.borderColor.resolve(state);
    const float borderWidth = style.borderWidth.resolve(state);
    
    // 绘制背景
    canvas.drawRoundRect(
        core::Rect::MakeWH(bounds_.width(), bounds_.height()),
        style.borderRadius,
        style.borderRadius,
        bgColor);
    
    // 绘制边框
    if (borderWidth > 0.0f) {
        canvas.drawRoundRect(
            core::Rect::MakeWH(bounds_.width(), bounds_.height()),
            style.borderRadius,
            style.borderRadius,
            borderColor,
            rendering::PaintStyle::Stroke,
            borderWidth);
    }
    
    // 绘制文本
    const float textX = (bounds_.width() - textWidth_) * 0.5f;
    const float textY = (bounds_.height() - textHeight_) * 0.5f + baseline_;
    canvas.drawText(label_, textX, textY, style.typography.button.fontSize, textColor);
}

WidgetState Button::currentState() const
{
    if (!enabled_) return WidgetState::Disabled;
    if (pressed_) return WidgetState::Pressed;
    if (hovered_) return WidgetState::Hovered;
    if (focused()) return WidgetState::Focused;
    return WidgetState::Normal;
}
```

### 5.2 动画切换实现

```cpp
// src/ui/ThemeManager.cpp

void ThemeManager::setTheme(const Theme& theme, bool animated)
{
    if (!animated) {
        currentTheme_ = theme;
        notifyThemeChange();
        return;
    }
    
    // 保存旧主题
    Theme oldTheme = currentTheme_;
    
    // 启动颜色过渡动画
    animationSink().animate(
        {
            .from = 0.0f,
            .to = 1.0f,
            .durationSeconds = 0.3,
            .easing = Easing::EaseInOut
        },
        [this, oldTheme, theme](float progress) {
            // 插值所有颜色
            currentTheme_.colors.primary = lerpColor(
                oldTheme.colors.primary,
                theme.colors.primary,
                progress);
            
            currentTheme_.colors.background = lerpColor(
                oldTheme.colors.background,
                theme.colors.background,
                progress);
            
            // ... 插值其他颜色
            
            notifyThemeChange();
        },
        [this, theme]() {
            // 动画完成，确保精确值
            currentTheme_ = theme;
            notifyThemeChange();
        });
}

core::Color lerpColor(const core::Color& a, const core::Color& b, float t)
{
    return core::Color::MakeRGBA(
        a.r() * (1.0f - t) + b.r() * t,
        a.g() * (1.0f - t) + b.g() * t,
        a.b() * (1.0f - t) + b.b() * t,
        a.a() * (1.0f - t) + b.a() * t);
}
```

## 九、性能考虑

### 9.1 样式缓存

```cpp
// Widget缓存解析后的样式
class Button : public Widget {
private:
    mutable std::optional<ButtonStyle> cachedStyle_;
    mutable uint64_t styleVersion_ = 0;
    
    const ButtonStyle& resolveStyle() const {
        uint64_t currentVersion = ThemeManager::instance().version();
        if (!cachedStyle_ || styleVersion_ != currentVersion) {
            cachedStyle_ = Widget::resolveStyle<ButtonStyle>();
            styleVersion_ = currentVersion;
        }
        return *cachedStyle_;
    }
};
```

### 9.2 避免重复计算

```cpp
// ✅ 好：只在主题切换时重新计算
void onThemeChange() {
    cachedStyle_.reset();
    markPaintDirty();
}

// ❌ 差：每帧都重新计算
void onDraw() {
    ButtonStyle style = resolveStyle<ButtonStyle>();  // 慢！
}
```

## 十一、常见问题

### Q1: 如何支持暗黑模式？

**A**: 注册两个主题，根据系统设置切换：
```cpp
ThemeManager::instance().registerTheme("light", Theme::light());
ThemeManager::instance().registerTheme("dark", Theme::dark());

// 根据系统设置切换
if (systemPrefersDarkMode()) {
    ThemeManager::instance().switchTheme("dark");
}
```

### Q2: 如何实现品牌色定制？

**A**: 使用`Theme::custom()`：
```cpp
ColorScheme brandColors = ColorScheme::custom(
    core::Color::MakeRGB(0, 122, 255));  // 品牌主色
Theme brandTheme = Theme::custom(brandColors);
ThemeManager::instance().setTheme(brandTheme);
```

### Q3: 样式会影响性能吗？

**A**: 不会，已优化：
- 样式解析结果会缓存
- 只在主题切换时重新计算
- 使用模板避免虚函数开销

### Q4: 如何支持CSS-like样式表？

**A**: 未来可以添加样式表解析器：
```cpp
// 未来功能
ThemeManager::instance().loadStyleSheet("styles.css");
```
