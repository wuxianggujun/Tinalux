## 一、动画系统架构

### 1.1 核心类

```cpp
// include/tinalux/ui/Animation.h

// 动画句柄
using AnimationHandle = std::uint64_t;

// 回调类型
using FrameCallback = std::function<void(double nowSeconds)>;
using AnimationCallback = std::function<void(float value)>;
using CompletionCallback = std::function<void()>;

// 缓动函数
enum class Easing {
    Linear,        // 线性
    EaseInOut,     // 缓入缓出
    EaseOutCubic,  // 三次缓出
};

// 补间动画选项
struct TweenOptions {
    float from = 0.0f;              // 起始值
    float to = 1.0f;                // 结束值
    double durationSeconds = 0.25;  // 持续时间（秒）
    bool loop = false;              // 是否循环
    bool alternate = false;         // 循环时是否往返
    Easing easing = Easing::EaseInOut;
};

// 动画接口
class AnimationSink {
public:
    virtual AnimationHandle requestAnimationFrame(FrameCallback callback) = 0;
    virtual AnimationHandle animate(
        const TweenOptions& options,
        AnimationCallback onUpdate,
        CompletionCallback onComplete = {}) = 0;
    virtual void cancelAnimation(AnimationHandle handle) = 0;
};
```

### 1.2 实现类

```cpp
// src/ui/RuntimeState.h

class AnimationScheduler : public AnimationSink {
public:
    // 实现AnimationSink接口
    AnimationHandle requestAnimationFrame(FrameCallback callback) override;
    AnimationHandle animate(
        const TweenOptions& options,
        AnimationCallback onUpdate,
        CompletionCallback onComplete = {}) override;
    void cancelAnimation(AnimationHandle handle) override;
    
    // 调度器方法
    void clear();
    bool hasActiveAnimations() const;
    bool tick(double nowSeconds);  // 每帧调用
};
```

### 1.3 主循环集成

```cpp
// src/app/Application.cpp

int Application::run()
{
    while (!impl_->window->shouldClose()) {
        // 1. 处理事件
        if (!impl_->uiContext.shouldRender()) {
            impl_->window->waitEventsTimeout(0.008);
        } else {
            impl_->window->pollEvents();
        }
        
        // 2. 更新动画（每帧tick）
        if (impl_->uiContext.tickAnimations(ui::animationNowSeconds())) {
            impl_->uiContext.noteAnimationTickUpdated();
        }
        
        // 3. 渲染
        if (impl_->uiContext.shouldRender()) {
            renderFrame();
        }
    }
}
```

### 2.1 获取AnimationSink

```cpp

ui::AnimationSink& sink = application.animationSink();
```

### 2.2 基础补间动画

```cpp
// 简单的淡入动画
AnimationHandle handle = animationSink().animate(
    {
        .from = 0.0f,
        .to = 1.0f,
        .durationSeconds = 0.3,
        .easing = Easing::EaseInOut
    },
    [this](float value) {
        // 每帧回调，value从0到1
        opacity_ = value;
        markPaintDirty();
    },
    [this]() {
        // 动画完成回调（可选）
        std::cout << "动画完成\n";
    }
);
```

### 2.3 取消动画

```cpp
// 保存动画句柄
AnimationHandle fadeAnim_;

// 启动动画
fadeAnim_ = animationSink().animate(/* ... */);

// 取消动画
animationSink().cancelAnimation(fadeAnim_);
fadeAnim_ = 0;
```

### 2.4 循环动画

```cpp
// 无限循环
animationSink().animate(
    {
        .from = 0.0f,
        .to = 1.0f,
        .durationSeconds = 1.0,
        .loop = true,
        .easing = Easing::Linear
    },
    [this](float value) {
        rotation_ = value * 360.0f;  // 旋转0-360度
        markPaintDirty();
    }
);

// 往返循环（0->1->0->1...）
animationSink().animate(
    {
        .from = 0.0f,
        .to = 1.0f,
        .durationSeconds = 0.5,
        .loop = true,
        .alternate = true,  // 往返
        .easing = Easing::EaseInOut
    },
    [this](float value) {
        scale_ = 0.9f + value * 0.1f;  // 0.9-1.0-0.9...
        markPaintDirty();
    }
);
```

### 2.5 requestAnimationFrame

```cpp
// 每帧回调（类似浏览器的requestAnimationFrame）
animationSink().requestAnimationFrame([this](double nowSeconds) {
    // 只执行一次
    updateSomething(nowSeconds);
});
```

```cpp
{
    const Theme& theme = resolvedTheme();
    
    // 基础背景色
    core::Color bgColor = theme.surface;

    }

    if (pressAlpha_ > 0.01f) {

    }
    
    canvas.drawRoundRect(

bool Button::onEvent(core::Event& event)
{
    switch (event.type()) {
    case core::EventType::MouseEnter:
        // 取消之前的动画
        if (hoverAnim_ != 0) {
            animationSink().cancelAnimation(hoverAnim_);
        }
        
        // 启动悬停动画：当前值 -> 1.0
        hoverAnim_ = animationSink().animate(
            {
                .from = hoverAlpha_,
                .to = 1.0f,
                .durationSeconds = 0.15,
                .easing = Easing::EaseOutCubic
            },
            [this](float value) {
                hoverAlpha_ = value;
                markPaintDirty();
            });
        return false;
        
    case core::EventType::MouseLeave:
        // 取消之前的动画
        if (hoverAnim_ != 0) {
            animationSink().cancelAnimation(hoverAnim_);
        }
        
        // 启动离开动画：当前值 -> 0.0
        hoverAnim_ = animationSink().animate(
            {
                .from = hoverAlpha_,
                .to = 0.0f,
                .durationSeconds = 0.15,
                .easing = Easing::EaseOutCubic
            },
            [this](float value) {
                hoverAlpha_ = value;
                markPaintDirty();
            });
        return false;
        
    case core::EventType::MouseButtonPress:
        // 取消之前的动画
        if (pressAnim_ != 0) {
            animationSink().cancelAnimation(pressAnim_);
        }
        
        // 启动按下动画：0 -> 1.0
        pressAnim_ = animationSink().animate(
            {
                .from = 0.0f,
                .to = 1.0f,
                .durationSeconds = 0.1,
                .easing = Easing::EaseOutCubic
            },
            [this](float value) {
                pressAlpha_ = value;
                markPaintDirty();
            });
        return true;
        
    case core::EventType::MouseButtonRelease:
        // 取消之前的动画
        if (pressAnim_ != 0) {
            animationSink().cancelAnimation(pressAnim_);
        }
        
        // 启动释放动画：当前值 -> 0.0
        pressAnim_ = animationSink().animate(
            {
                .from = pressAlpha_,
                .to = 0.0f,
                .durationSeconds = 0.2,
                .easing = Easing::EaseOutCubic
            },
            [this](float value) {
                pressAlpha_ = value;
                markPaintDirty();
            });
        
        // ... 触发onClick回调
        return shouldClick || inside;
        
    // ... 其他事件处理
    }
}

// 辅助函数：颜色混合
core::Color blendColors(core::Color base, core::Color overlay, float alpha)
{
    return core::Color::MakeRGBA(
        base.r() * (1.0f - alpha) + overlay.r() * alpha,
        base.g() * (1.0f - alpha) + overlay.g() * alpha,
        base.b() * (1.0f - alpha) + overlay.b() * alpha,
        base.a());
}
```

## 四、涟漪效果实现

### 4.1 RippleEffect结构

```cpp
// Button.h
struct RippleEffect {
    core::Point center;    // 点击位置
    float radius = 0.0f;   // 当前半径
    float alpha = 1.0f;    // 透明度
    bool active = false;
};

class Button : public Widget {
private:
    RippleEffect ripple_;
    AnimationHandle rippleAnim_ = 0;
};
```

### 4.2 绘制涟漪

```cpp
void Button::onDraw(rendering::Canvas& canvas)
{
    // ... 绘制背景
    
    // 绘制涟漪效果
    if (ripple_.active && ripple_.alpha > 0.01f) {
        const Theme& theme = resolvedTheme();
        core::Color rippleColor = theme.primary;
        rippleColor.setAlpha(ripple_.alpha * 0.3f);  // 半透明
        
        canvas.drawCircle(
            ripple_.center,
            ripple_.radius,
            rippleColor);
    }
    
    // ... 其余绘制
}
```

### 4.3 启动涟漪动画

```cpp
bool Button::onEvent(core::Event& event)
{
    if (event.type() == core::EventType::MouseButtonPress) {
        const auto& mouseEvent = static_cast<const core::MouseButtonEvent&>(event);
        
        // 记录点击位置（本地坐标）
        ripple_.center = globalToLocal(core::Point::Make(
            static_cast<float>(mouseEvent.x),
            static_cast<float>(mouseEvent.y)));
        ripple_.active = true;
        
        // 计算最大半径（到最远角的距离）
        const float w = bounds_.width();
        const float h = bounds_.height();
        const float maxRadius = std::sqrt(w * w + h * h);
        
        // 取消之前的涟漪
        if (rippleAnim_ != 0) {
            animationSink().cancelAnimation(rippleAnim_);
        }
        
        // 启动涟漪扩散动画
        rippleAnim_ = animationSink().animate(
            {
                .from = 0.0f,
                .to = maxRadius,
                .durationSeconds = 0.4,
                .easing = Easing::EaseOutCubic
            },
            [this, maxRadius](float value) {
                ripple_.radius = value;
                ripple_.alpha = 1.0f - (value / maxRadius);  // 逐渐淡出
                markPaintDirty();
            },
            [this]() {
                ripple_.active = false;  // 动画完成
            });
    }
}
```

## 七、性能考虑

### 7.1 动画触发重绘

```cpp
// AnimationScheduler::tick 返回是否有更新
bool updated = animationScheduler.tick(nowSeconds);

// Application主循环检查
if (updated) {
    uiContext.noteAnimationTickUpdated();  // 触发重绘
}
```

**优化**：只有动画活跃时才重绘，无动画时使用waitEventsTimeout节省CPU。

### 7.2 避免过多动画

- 同一时间不要启动过多动画（建议<20个）
- 取消不需要的动画
- 使用AnimatedWidget基类统一管理

## 九、常见问题

### Q1: Widget如何访问AnimationSink？

**A**: 当前需要架构改进，建议：
1. Widget基类添加animationSink()虚方法
2. 或通过全局Application实例访问
3. 或在UIContext渲染时通过RuntimeState访问

### Q2: 动画会影响性能吗？

**A**: 不会，已优化：
- 只在有动画时触发重绘
- 无动画时使用waitEventsTimeout节省CPU
- AnimationScheduler高效管理所有动画

### Q3: 如何实现复杂的动画序列？

**A**: 使用完成回调链式调用：
```cpp
animationSink().animate(
    {/* 动画1 */},
    [](float v) { /* 更新1 */ },
    [this]() {
        // 动画1完成，启动动画2
        animationSink().animate(
            {/* 动画2 */},
            [](float v) { /* 更新2 */ });
    });
```
