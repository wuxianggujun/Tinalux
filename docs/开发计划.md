# Tinalux 开发计划

## 阶段一：渲染闭环 ✅ 已完成

**目标**：GLFW 窗口 + Skia 绘制跑通

- [x] CMake 多模块构建（星形依赖：App→Platform/Rendering/UI→Core）
- [x] GLFW 3.4 FetchContent 集成
- [x] Skia 配置（cmake/SkiaConfig.cmake，Debug/Release 双目标）
- [x] Core: Event 基类 + 全部事件类型（13 种）+ EventDispatcher
- [x] Platform: Window 抽象接口 + GLFWWindow 完整实现
  - 8 个 GLFW 回调（resize/close/focus/key/mouse/cursor/scroll/char）

  - GLFW 生命周期引用计数管理
- [x] Rendering: initSkia/shutdownSkia + createGLContext(GLGetProcFn) + createWindowSurface + flushFrame
  - GL 函数指针注入，不依赖 Platform/GLFW
- [x] App: Application（Pimpl 模式）init/run/shutdown + 渲染循环
  - 脏标记跳帧（无变化时 sleep 8ms）
- [x] 删除旧 Qt 文件
